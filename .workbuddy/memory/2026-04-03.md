### 功能说明
1. **26 个 OpenCLI 内置工具**（权限: limited）:
   - Hacker News: top, new, best, ask, show, jobs, search
   - Stack Overflow: hot, search
   - DEV.to: top, tag
   - Lobsters: hot, newest, active
   - V2EX: hot, latest
   - Reddit: hot, search
   - Product Hunt: today
   - Steam: top-sellers
   - ArXiv: search
   - HuggingFace: trending
   - Wikipedia: search, summary, trending
   - 36kr: hot

2. **web_cli 通用代理工具** — 通过平台代理 API 调用任意 CLI 命令
   - 参数: site (必填), command (必填), query (搜索关键词), limit (条数)

3. **平台代理 API**:
   - `GET /api/cli/:site/:cmd?q=xxx&limit=20` — 执行 CLI 命令
   - `GET /api/cli` — 列出所有可用命令
   - 18 个服务端代理命令实现

## 阶段三架构复盘
- 当前实现属于“过渡版”：平台路由本质上还是 `fetch` handler 映射，尚未实现真正的三引擎分发（fetch / browser / auth）。
- 确认后续正确方向是平台中心化 `web_cli`：Agent 端尽量只保留通用 HTTP 调用工具，浏览器池、登录态、缓存、限流、审计放在平台侧。
- 若进入下一步开发，优先重构平台侧命令注册表与执行引擎抽象，再接浏览器池和塘主登录态注入。
